Top Navigation Bar
1. Left Corner Branding

    If the URL is v4l.app:
    Display the V4L Application Logo.

    If the URL contains a subdomain (e.g., nbs.v4l.app):
    Display the logo of the organization corresponding to that subdomain (e.g., nbs organization logo).

2. Right-Side Navigation Items
    2.1 My

        User profile access.

        Links to user account settings and personal dashboard.

    2.2 Organizations

        Navigates to /organizations.

        Provides a dashboard for managing all organizations associated with the user.

    2.3 Vacancies

        Shows open job positions.

        If URL is v4l.app:
        Display vacancies from all organizations in the system.

        If URL has a subdomain (e.g., nbs.v4l.app):
        Show vacancies only for that specific organization.

    2.4 Market

        Shows goods, services, needs, and wants for purchase or rent.

        If URL is v4l.app:
        View marketplace listings from all organizations.

        If URL contains a subdomain:
        Show marketplace listings only for the associated organization.

Page-Level Navigation Structure
    /organizations

        Displays list of all organizations that the user is associated with.

        Shows department dashboards for quick access.

    /organizations/facilities

        Facilities management dashboard. Also manages the other departments under Facility.


        Quick links to:

            Branches

            Buildings

            Workstations

            and Other Departments in the Facility Department

    /organizations-facilities-branches

        Branch management dashboard.

        Displays list of branches with options to:

            Create

            Read

            Update

            Delete
